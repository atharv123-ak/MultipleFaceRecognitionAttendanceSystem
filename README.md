# MultipleFaceRecognitionAttendanceSystem
A real-time multi-face recognition attendance system using Python, OpenCV, and the Face Recognition library with Firebase integration for cloud-based attendance tracking.
📌 Project Overview
The Multiple Face Recognition Attendance System is an automated attendance management system developed using Python and computer vision techniques. The system detects and recognizes multiple faces in real time using a webcam and marks attendance automatically.
To ensure real-time data storage and accessibility, the attendance records are updated and stored in Firebase Realtime Database. This eliminates the need for manual attendance marking and improves accuracy and efficiency.
🎯 Objectives
Automate the traditional attendance process using face recognition technology
Reduce manual errors in attendance management
Enable real-time attendance tracking using Firebase
Build a scalable system that can detect and recognize multiple faces simultaneously

| -------------------------- | ---------------------------------------- |

| OpenCV                     | Face detection and image processing      |
| Face Recognition Library   | Face encoding and recognition            |
| Firebase Realtime Database | Cloud-based real-time attendance storage |
